## Overview

Protocol for fragmenting gDNA using Diagenode Megaruptor® 3 for subsequent Ligation Sequencing Kit library preparation.[1] Fragmentation benefits include reduced library input requirements, increased read length N50, and reduced pore blocking.[1] Optional size selection before shearing helps remove shortest fragments.[1]

## Key Parameters

- **Input**: 2000 ng gDNA in 100 µl nuclease-free water[1]
- Use Megaruptor 3 Shearing Kit (cat. E07010003)[1]
- QC with Agilent FEMTO Pulse or equivalent[1]
