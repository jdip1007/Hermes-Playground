## Overview

Protocol for fragmenting HMW genomic DNA using Covaris g-TUBE centrifugal force shearing.[1] Recommended for lower input samples (100 ng–500 ng) where fragmentation increases molecule count and throughput.[1] Standard protocol uses >4 µg, but Oxford Nanopore validated 100–1000 ng in 49 µl fragments equivalently.[1]

## Key Parameters

- **Microfuge**: Eppendorf 5424 at 6000 rpm yields ~8 kb average Lambda DNA fragments[1]
- Different centrifuges/samples require speed optimization — refer to g-TUBE literature[1]
- No beads required — purely centrifugal force shearing[1]
