## Overview

Method to extract HMW genomic DNA from green anole lizard (*Anolis carolinensis*) tail as an example of reptile tissue.[1] Tail collected via autotomy (lizard drops tail when manipulated), stored in 96–100% ethanol for one week at room temperature before extraction. Also works with fresh tissue.[1] Uses QIAGEN Blood and Cell Culture DNA Midi Kit with semi-selective precipitation size selection before MinION sequencing.[1]

## Key Steps

- Start with 100 mg lizard tail tissue (ethanol-preserved or fresh)[1]
- Extract with QIAGEN Blood and Cell Culture DNA Midi Kit + RNase A + Proteinase K[1]
- Size select 3 µg using semi-selective precipitation before library prep[1]
