## Overview

Method to extract HMW genomic DNA from human saliva pooled from multiple individuals.[1] Saliva stored in Isohelix GeneFiX™ Saliva DNA Collection Device, extracted using QIAGEN Blood and Cell Culture DNA Midi Kit followed by semi-selective DNA precipitation for size selection before MinION sequencing.[1]

## Key Steps

- Collect 1 ml saliva, store at room temperature in Isohelix GeneFiX device[1]
- Extract with QIAGEN Blood and Cell Culture DNA Midi Kit + RNase A + Proteinase K[1]
- Size select using semi-selective precipitation (2.5% w/v PVP 360,000, 1.2 M NaCl)[1]
