## Overview

Method to extract HMW genomic DNA from western clawed frog (*Xenopus tropicalis*) muscle as an example of amphibian tissue.[1] Uses QIAGEN Blood and Cell Culture DNA Midi Kit with semi-selective DNA precipitation for size selection before PromethION sequencing.[1]

## Key Steps

- Start with 100 mg frozen frog muscle tissue (stored at −80°C)[1]
- Homogenize using QIAGEN TissueRuptor II[1]
- Extract with QIAGEN Blood and Cell Culture DNA Midi Kit + RNase A + Proteinase K[1]
- Size select using semi-selective precipitation (2.5% w/v PVP 360,000, 1.2 M NaCl)[1]
