## Overview

Oxford Nanopore's cfDNA information repository documenting internally developed methods, know-how documents and best practices for nanopore sequencing of cell-free DNA.[1] Applications include cancer detection and tissue-of-origin analysis.[1]

## Key Resources

- **Singleplex protocol**: Ligation Sequencing Kit V14 (SQK-LSK114) — single human cfDNA sample from fresh blood or plasma[1]
  - Typical output: ~50 Gb aligned data (~15× coverage) per PromethION flow cell with ~15 ng input[1]
  - Average sequencing output: 70 Gb, primary aligned data: 52 Gb across 4 replicates[1]
