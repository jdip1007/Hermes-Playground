## Overview

Method to extract HMW genomic DNA from buffy coat isolated from rabbit blood as an example of mammalian blood.[1] Buffy coat is the white blood cell and platelet fraction following centrifugation of anticoagulated blood.[1] Uses QIAGEN MagAttract HMW DNA kit with SPRI bead size selection (3 µg) before MinION sequencing with Ligation Sequencing Kit.[1]

## Key Steps

- Start with 10 ml rabbit blood in anticoagulant[1]
- Isolate buffy coat via Ficoll-Paque PREMIUM density gradient centrifugation[1]
- Extract DNA using QIAGEN MagAttract HMW DNA kit[1]
- Size select 3 µg extracted DNA using SPRI beads before library prep[1]
