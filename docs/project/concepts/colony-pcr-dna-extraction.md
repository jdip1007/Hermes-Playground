## Overview

Method to extract and prepare genomic and plasmid DNA directly from bacterial colonies, verified with Gram-negative *E. coli*.[1] Colony picked from plate, treated with Proteinase K, then prepared using Rapid PCR Barcoding Kit for GridION sequencing.[1]

## Key Steps

- Pick bacterial colony into 1.5 ml LoBind tube[1]
- Treat with Thermolabile Proteinase K (optional but recommended)[1]
- Prepare library using Rapid PCR Barcoding Kit + SFB Expansion[1]
- AMPure XP bead clean-up before loading[1]
