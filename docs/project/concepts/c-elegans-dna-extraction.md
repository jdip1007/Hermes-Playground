## Overview

Method to extract HMW genomic DNA from *C. elegans* strain SX3254 (descendent from N2 wild type).[1] Worms grown in NGM media with *E. coli* food source, harvested with M9 buffer.[1] Extraction uses QIAGEN Puregene Cell Kit with semi-selective DNA precipitation for size selection before MinION sequencing.[1]

## Key Steps

- Harvest worms from one plate using M9 buffer (0.85 M Na₂HPO₄, 0.4 M KH₂PO₄, 0.3 M NaCl, 1 mM MgSO₄)[1]
- Store pellet at −20°C or −80°C[1]
- Extract using QIAGEN Puregene Cell Kit + RNase A + Proteinase K[1]
- Size select using semi-selective DNA precipitation (2.5% w/v PVP 360,000, 1.2 M NaCl)[1]
