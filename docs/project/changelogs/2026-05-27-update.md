ABC 契约（`base.py:65-122`）：

```python
class DashboardAuthProvider(ABC):
    name: str = ""           # 稳定 lowercase 标识符
    display_name: str = ""   # 登录页面板用

    @abstractmethod
    def start_login(self, *, redirect_uri: str) -> LoginStart: ...
    @abstractmethod
    def complete_login(self, *, code, state, code_verifier, redirect_uri) -> Session: ...
    @abstractmethod
    def verify_session(self, *, access_token: str) -> Optional[Session]: ...
    @abstractmethod
    def refresh_session(self, *, refresh_token: str) -> Session: ...
    @abstractmethod
    def revoke_session(self, *, refresh_token: str) -> None: ...
```

失败语义被 middleware 映射为 HTTP 状态码：`ProviderError → 503`、`InvalidCodeError → 400`、`RefreshExpiredError →` 清 cookie + `302 → /login`。

### 1.3 WS upgrade 单次 ticket 机制

浏览器 WebSocket upgrade 不能带 `Authorization` 头。`ws_tickets.py` 提供：
- `mint_ticket(*, user_id, provider)` — 32-byte URL-safe random + 30s TTL + 单次性（成功 consume 即弹出）

- SPA 流程：fetch `/api/auth/ws-ticket` → 拿 token → `wss://...?ticket=<token>` upgrade
- 4 个 WS 端点全部经 `_ws_auth_ok` 校验

## 4. security-guidance 插件 — 25 条 dangerous-pattern 警告

`plugins/security-guidance/`（5 文件、627 行）：commit `249534e` (#33131)，作者 Teknium。

### 4.1 设计哲学

非阻塞为默认（`SECURITY_GUIDANCE_BLOCK=1` 升级阻塞、`SECURITY_GUIDANCE_DISABLE=1` killswitch）：

> Patterns 有非平凡的假阳率（`eval(` 在 tokenizer / `yaml.load` 已被 `yaml.SafeLoader` 包裹 / ECB 在测试 fixture）。阻塞会迫使每个假阳触发审批中断。警告是 layer 1 的正确严重度 — agent 读警告，要么修代码要么文档化解释为何安全。

### 4.2 25 条 SECURITY_PATTERNS（`patterns.py:53-368`）

涵盖：
- **Python**: `pickle.load` / `yaml.load` (no SafeLoader) / `eval(` / `os.system` / `subprocess(..., shell=True)` / `torch.load`（无 `weights_only=True`）/ XXE-prone `xml.etree` + `minidom` parser
- **Node.js**: `child_process.exec` / `dangerouslySetInnerHTML` / `innerHTML` / `outerHTML` / `document.write` / `insertAdjacentHTML`
- **Crypto**: `crypto.createCipher`（无 IV） / AES ECB / TLS verification disabled（`verify=False`）
- **HTML**: `<script src="//...">` 无 SRI
- **CI/CD**: GitHub Actions `${{ github.event.* }}` injection

### 4.3 Hook 接入

`plugin.yaml`:
```yaml
hooks:
  - transform_tool_result
  - pre_tool_call
```

`__init__.py` 扫描 `write_file` / `patch` / `skill_manage`（write 与 patch mode）的内容参数，命中即在 JSON 工具结果 string 后追加 `⚠️ Security warning` 块。文件已写入，警告在下回合走回模型让其自我纠正。

### 4.4 来源与授权

`patterns.py` 是 Anthropic `claude-plugins-official/plugins/security-guidance/hooks/patterns.py @ 0bde168`（2026-05-26）的 Apache-2.0 verbatim fork。`LICENSE` + `NOTICE` 保留归属。Hermes 端 plugin glue（`__init__.py` / `plugin.yaml` / `README.md` / 测试）是原创。
