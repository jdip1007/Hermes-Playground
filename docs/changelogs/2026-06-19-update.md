# Changelog — 2026-06-19

## New Pages

### Concepts

- **human-cell-line-dna-puregene-extraction** — Human Cell Line DNA Extraction using QIAGEN Puregene Cell Kit (Oxford Nanopore Protocol v3): 14-step protocol yielding 20–30 µg HMW DNA for Nanopore sequencing. Source: Oxford Nanopore Technologies protocol document (v3, Dec 2022).
- **human-cell-line-dna-genomic-tip-extraction** — Human Cell Line DNA Extraction using QIAGEN Genomic-tip / Blood & Cell Culture DNA Maxi Kit (Oxford Nanopore Protocol v3): Large-scale protocol (1 × 10⁸ cells) yielding 400–450 ng/µl HMW DNA. Source: Oxford Nanopore Technologies protocol document (v3, Aug 2023).
- **gram-positive-bacterial-dna-puregene-extraction** — Gram-Positive Bacterial DNA Extraction using QIAGEN Puregene Cell Kit + Lytic Enzyme Solution (Oxford Nanopore Protocol v2): 19-step protocol for gram-positive organisms requiring enzymatic cell wall digestion. Source: Oxford Nanopore Technologies protocol document (v2, Jan 2023).
- **gram-positive-bacterial-dna-midi-kit-extraction** — Gram-Positive Bacterial DNA Extraction using QIAGEN Blood & Cell Culture DNA Midi Kit (Oxford Nanopore Protocol): Large-scale protocol (1 × 10¹¹ cells) yielding 125–140 µg HMW DNA with spooling instead of centrifugation to preserve fragment length. Source: Oxford Nanopore Technologies protocol document.
- **rabbit-muscle-dna-genomic-tip-extraction** — Rabbit Muscle DNA Extraction using QIAGEN Genomic-tip / Blood & Cell Culture DNA Midi Kit (Oxford Nanopore Protocol v3): Protocol for up to 100 mg muscle tissue yielding 30–50 µg HMW DNA with overnight Proteinase K digestion at 50°C. Source: Oxford Nanopore Technologies protocol document (v3, Oct 2025).
- **rabbit-muscle-dna-atl-buffer-extraction** — Rabbit Muscle DNA Extraction using QIAGEN Genomic-tip with ATL Buffer (Oxford Nanopore Protocol v2): Two-stage Proteinase K digestion protocol yielding 10–20 µg HMW DNA from up to 100 mg tissue. Source: Oxford Nanopore Technologies protocol document (v2, Aug 2023).
- **rabbit-skin-dna-genomic-tip-extraction** — Rabbit Skin DNA Extraction using QIAGEN Genomic-tip with G2 buffer (Oxford Nanopore Protocol v2): Protocol for up to 100 mg skin tissue yielding 30–60 µg HMW DNA with overnight Proteinase K digestion. Source: Oxford Nanopore Technologies protocol document (v2, Aug 2023).
- **rabbit-lung-dna-atl-buffer-extraction** — Rabbit Lung DNA Extraction using QIAGEN Genomic-tip with ATL Buffer (Oxford Nanopore Protocol v2): Two-stage Proteinase K digestion protocol yielding ~50 µg HMW DNA from up to 100 mg lung tissue. Source: Oxford Nanopore Technologies protocol document (v2, Aug 2023).
- **rabbit-lung-dna-g2-buffer-extraction** — Rabbit Lung DNA Extraction using QIAGEN Genomic-tip with G2 Buffer (Oxford Nanopore Protocol v2): Single overnight Proteinase K digestion protocol yielding ~50 µg HMW DNA from up to 100 mg lung tissue. Source: Oxford Nanopore Technologies protocol document (v2, Aug 2023).
- **rabbit-skin-dna-atl-buffer-extraction** — Rabbit Skin DNA Extraction using QIAGEN Genomic-tip with ATL Buffer (Oxford Nanopore Protocol v2): Two-stage Proteinase K digestion protocol yielding 10–20 µg HMW DNA from up to 100 mg skin tissue. Source: Oxford Nanopore Technologies protocol document (v2, Aug 2023).
