# Post-v0.13.0 同步 — 2026-05-12

v0.13.0 发布后约一周的增量。下一次正式 release 之前的主要变化。

## 主要 feat

### 1. Cross-session 1h prefix cache for Claude（PR #23828）

详见 [[prompt-caching-optimization]] 全文。`agent/prompt_caching.py` 新增 `apply_anthropic_cache_control_long_lived` + `mark_tools_for_long_lived_cache`，专门为 Claude on Anthropic / OpenRouter / Nous Portal 三条路径设计 4 断点布局：

| 断点 | 位置 | TTL |
|------|------|-----|
| tools[-1] | 工具数组末尾 | **1h** |
| system block[0] | 系统消息稳定前缀块 | **1h** |
| 倒数第 2 / 1 条非系统消息 | 滚动窗口 | 5m |

第一次会话写入 1h 缓存，第二次会话**冷启动**即命中。先决：系统消息必须被 `prompt_builder` 切成有序 content blocks，block[0] 字节稳定。

后续修复 PR #24151：把 Nous Portal **Qwen** 也路由到 Portal-Claude cache pathway，避免错过 1h 缓存。

### 2. `feat(agent): per-turn file-mutation verifier footer`（PR #24498）

`run_agent.py:5453-5478` 新增 `_file_mutation_verifier_enabled()` + `_format_file_mutation_failure_footer()`。每轮结束时若 `write_file` / `patch` 有失败（语法 / 路径 / 权限），自动在助手回复末尾贴上 footer，列出最多 10 条失败路径 + 首条错误。

- 配置：`display.file_mutation_verifier` (bool, 默认 true)
- env 覆盖：`HERMES_FILE_MUTATION_VERIFIER=0`

跟 v0.13 的 post-write delta lint 配套：lint 在工具层发现错误，verifier 在回复层暴露给用户。

### 3. `feat(security): supply-chain advisory checker + lazy-install framework + tiered install fallback`（PR #24220）

新文件：`hermes_cli/security_advisories.py`、`tools/lazy_deps.py`。

- 启动时（CLI、Gateway）扫描已知 supply-chain advisory（举例：May 2026 Mini Shai-Hulud worm），有命中时显眼提醒
- `security.allow_lazy_installs` 配置（默认 true）控制是否允许「按需 lazy install」依赖
- tiered install fallback：依赖装失败时分级降级，而不是死在 traceback

### 4. `feat: expose HERMES_SESSION_ID to agent tools via ContextVar + env`（PR #23847）

`gateway/session_context.py:58` 新增 `_SESSION_ID` ContextVar，配合 `run_agent.py:1913` 设置 `os.environ["HERMES_SESSION_ID"]`，让工具子进程（shell / skill 脚本 / mcp_serve）能拿到当前 session id。

`hermes_cli/config.py:1146` 同时支持 SKILL.md 里用 `${HERMES_SESSION_ID}` 占位符。

### 5. `Use Nous Portal as model metadata authority`（PR #24502, #24509, e2b713c）

- model picker / metadata resolution 把 **Nous Portal** 视为权威 catalog（替代 OpenRouter 兜底）
- known provider 跳过 OpenRouter 探测（PR e2b713c）
- paid recommendations 取 Portal ∪ static 列表并集（PR #24509）
- Portal-flagged free model 即使本地 curated list 过期也显示（PR #24082）

### 6. `feat(computer-use): refresh cua-driver on hermes update + add install --upgrade`（PR #24063）

cua-driver（computer use）现在跟随 `hermes update`，并提供 `hermes install --upgrade` 显式升级路径。

### 7. `feat(ui-tui): resolve markdown links to readable page titles`（PR #24013）

TUI 显示 markdown link 时解析远端 `<title>`，把 `https://example.com/foo` 渲染成实际页面标题。

### 8. `add client marker tag on aux inference requests`（94d9db7）

辅助模型请求加上 client marker tag，方便 Nous Portal / 上游做请求归因。
