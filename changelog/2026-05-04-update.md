## 1. 自治 Curator 与自我提升循环

### Curator 作为后台 agent
- 跑在 gateway cron-ticker 上，**默认 7 天周期**。源：`agent/curator.py:56` `DEFAULT_INTERVAL_HOURS = 24 * 7`
- 每轮报告 `logs/curator/run.json` + `REPORT.md`
- 归档技能用模型 + 启发式分类为 **consolidated vs pruned**
- `hermes curator status` 按使用频次排序（most-used / least-used）
- 统一在 `auxiliary.curator.{provider, model, timeout, ...}`，可在 `hermes model` / dashboard 配置。源：`agent/curator.py:1483, 1499, 1551`
- `bump_use()` 接入 skill 调用 + preload + skill_view（#17932）
- defense-in-depth：bundled / hub-installed / pinned 三类技能受保护

### 自我提升循环（background review fork）
- **class-first 评分**（rubric-based 而非 free-form）（#16026）
- **active-update 偏置**：优先更新 agent 刚加载的 skill；处理 `references/` 和 `templates/` 子文件（#17213）
- **fork 继承父 agent 的 live runtime**：provider/model/credentials 真正传递（#16099）
- **scoped toolsets**：仅 memory + skills，不能跑 shell/web（#16569）
- 干净 shutdown（#16204）+ 干净 context（排除上一轮的 tool 消息，#15057）

## 6. Cron 增强

### `no_agent` 模式（脚本即任务）
- 源：`cron/jobs.py:438 no_agent: bool = False`，`cron/scheduler.py:857 # no_agent short-circuit`
- `no_agent=True` **跳过 LLM**：脚本就是任务，stdout 原样投递；空 stdout 视为静默成功
- 必须配 `script`（无脚本拒绝）；`prompt` 字段被忽略
- 模块顶层延迟导入，watchdog tick 不为 AIAgent / SessionDB 付出冷启动开销

### 其他
- `croniter` 升为核心依赖（#17577）
- 每个 job 支持 `workdir`（#15110）+ `context_from`（#15606）
- `${VAR}` 在 cron 任务运行时从 `config.yaml` 展开（#15890）
- 并发跑同一 job 单独写状态、规避 race（`1c7f47a`）

## 8. 安全 / 可靠性

### Secret redaction 默认 OFF（#16794）
源：`agent/redact.py:64`：

```python
_REDACT_ENABLED = os.getenv("HERMES_REDACT_SECRETS", "").lower() in ("1","true","yes","on")
# OFF by default — opt in via security.redact_secrets: true
```

历史上 redaction 把 patch / API payload 的"key-shaped"子串改坏（patch corruption），所以 flip 到 off。`browser-tool-architecture` 的快照仍**显式** `force=True` 脱敏。

### 系统标记重命名（#16114）
所有 `[SYSTEM:` 标记改为 `[IMPORTANT:`，绕过 Azure content filter。

### Hardline blocklist（#15878）
某些不可恢复命令（`rm -rf /`、`> /dev/sda`）**直接拒绝**，连 manual 模式都不弹审批。`DANGEROUS_PATTERNS` / `HARDLINE_PATTERNS` 预编译（#17206）。

### 审批模式扩大覆盖（v0.12.0 之后 #69dd0f7）
敏感写目标加入 shell rc 文件、credential 文件。

### Vision 缓存改 HERMES_HOME（#17719）
不再放 `cwd`。
