### 3.6 `/undo` vs Checkpoint 关系

- `/undo` 是**逻辑层**回退（消息列 active=0，audit 可恢复）。
- [[checkpoints-architecture]] 是**文件系统层**回退（shadow git store）。
- checkpoint-rollback 调 `undo_last(n=..., prefill=False)` 把消息一并 truncate。

[[cli-architecture]] / [[messaging-gateway-architecture]] 中 `/undo` 描述需从"单回合硬截"更新为"N 回合软删 + prefill"。

## 4. Curator built-in 修剪 + 全 skill 用量遥测
