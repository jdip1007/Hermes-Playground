# Kanban Architecture

> Kanban-based task board and orchestration system

## Overview

Kanban is a task management and orchestration system in Hermes Agent that provides:
- Visual task boards (To Do, In Progress, Done)
- Task lifecycle management
- Worker pool coordination
- State persistence

## Components

- Board manager
- Column state machine
- Worker dispatcher
- Task claim/release
