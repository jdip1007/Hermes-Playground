# Multi-Agent Kanban

> Multi-agent coordination via Kanban boards

## Overview

Multi-agent Kanban extends the Kanban system for coordinating multiple agents:
- Agent assignment to tasks
- Inter-agent dependencies
- Parallel execution
- Collision avoidance

## Features

- Agent-capability matching
- Task dependency graphs
- Parallel execution scheduling
- State synchronization
