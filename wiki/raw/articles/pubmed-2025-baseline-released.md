        <h1>PubMed 2025 Baseline Released</h1>
 

<a href="https://techbull.nlm.nih.gov/vivisimo/cgi-bin/query-meta?v%3Aproject=technical-bulletin-date&sortby=Date-decending&query=&binning-state=product==MeSH"><button class="usa-tag padding-y-05 padding-x-1">Mesh</button></a> <a href="https://techbull.nlm.nih.gov/vivisimo/cgi-bin/query-meta?v%3Aproject=technical-bulletin-date&sortby=Date-decending&query=&binning-state=product==PubMed"><button class="usa-tag padding-y-05 padding-x-1">Pubmed</button></a> 

        <p class="text-base">PubMed 2025 Baseline Released. NLM Tech Bull. 2025 Jan-Feb;(<a href="/pubs/techbull/jf25/jf25_issue_cover.html">462</a>):e1.</p>


         
        <p class="border-bottom-1px border-primary grid-col-6 padding-bottom-3 margin-bottom-0">January 14, 2025 [posted] 
       


</p>
          
      </div>
    </div>
    <div class="grid-row grid-gap">
 
   
  
<div class="desktop:grid-col-6 tablet:grid-offset-1 padding-y-3 tablet:grid-col-12 mobile:grid-col-12">
 
   

       <p>The National Library of Medicine (NLM) has released the 2025 production year PubMed baseline files. The complete 2025 baseline consists of files pubmed25n0001 through pubmed25n1274.xml. In addition to the XML files there are corresponding MD5 checksum files for each XML export file.</p>
<p>The first set of 2025 PubMed Update files have been posted. The first 2025 update file is pubmed25n1275.xml. Regular daily updates have resumed.</p>
<p>NOTE: All baseline files must be downloaded and processed PRIOR to loading the first and subsequent update files.</p>
<p>DTD Changes: On September 28, 2024, NLM announced the release of the <a href="https://dtd.nlm.nih.gov/ncbi/pubmed/out/pubmed_250101.dtd">2025 PubMed DTD</a>.</p>
<p>Additions/Changes since pubmed_240101.dtd:</p>
<ol>
<li>Updated DescriptorName model to allow for optional AutoHM="Y" attribute.</li>
<li>Updated QualifierName model to allow for optional AutoHM="Y" attribute.<br />The&nbsp;<a href="https://web.archive.org/web/20250202083359/https:/dtd.nlm.nih.gov/ncbi/pubmed/doc/out/250101/att-AutoHM-c.html">AutoHM</a>&nbsp;attribute will be used to distinguish between Descriptors/Qualifiers that have been added by curators or MTIX based on information in the article text (attribute not present), and Descriptors/Qualifiers that have been added automatically after a supplementary concept record (SCR) was indexed (AutoHM="Y"). Learn more about&nbsp;<a href="https://web.archive.org/web/20250202083359/https:/www.nlm.nih.gov/mesh/intro_record_types.html">MeSH record types</a>.</li>
<li>Updated&nbsp;<a href="https://web.archive.org/web/20250202083359/https:/dtd.nlm.nih.gov/ncbi/pubmed/doc/out/250101/att-IndexingMethod-c.html">IndexingMethod</a> attribute to include allowed value "Manual" for fully human-indexed citations. This attribute has been retroactively applied to manually indexed citations in PubMed.</li>
</ol>
<p>Please see <a href="https://dtd.nlm.nih.gov/ncbi/pubmed/doc/out/250101/index.html">pubmed_250101.dtd Documentation</a> for annotations and examples for elements and attributes defined in the PubMed DTD. Learn more about <a href="https://pubmed.ncbi.nlm.nih.gov/download/">downloading PubMed data</a>.</p>
<p>If you have questions regarding this process, please write to the <a href="https://support.nlm.nih.gov/support/create-case/">NLM Help Desk</a>.</p>
